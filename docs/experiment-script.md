# Experiment Script

## Purpose
This script gives each tester the same set of tasks so we can measure whether students can complete the core Student Apartment Finder workflow without help.

## Before You Start (facilitator notes)
- Open the prototype and make sure listings load.
- Tell the tester: "Think aloud as you go. There are no wrong answers — we are testing the app, not you."
- Do not help unless the tester is fully stuck for over 30 seconds.

## Tester Tasks

| Task No. | Task Instruction | Related Requirement | Screen | Success = |
|---|---|---|---|---|
| 1 | Open the home page and tell us what this website is for. | FR-01 | Home | Tester correctly describes it as finding student apartments near campus |
| 2 | Find an apartment near Rangsit University under 6,000 THB. | FR-06 | Apartments + filters | Tester uses search/filter and finds a matching room |
| 3 | Open one apartment and tell us its price, distance, and status. | FR-07, FR-08 | Detail view | Tester reads price, distance, and availability correctly |
| 4 | Show us on the map how far it is from campus. | FR-07 | Map | Tester locates the listing on the map |
| 5 | Find how to contact the owner of that apartment. | FR-02 | Detail view | Tester reaches the contact-owner step |
| 6 | Save this apartment to look at later. | FR-05 | Save button | Tester saves the listing and sees confirmation |
| 7 | Find a roommate post on the roommate board. | FR-05 | Roommate board | Tester opens the roommate board and views a post |

## After the Tasks
Ask the tester to complete the feedback form (`docs/feedback-form.md`).

## What the Facilitator Records
- Whether each task was completed (Yes / No / With help)
- Time taken for each task
- Any point of confusion or hesitation
- Direct tester quotes
